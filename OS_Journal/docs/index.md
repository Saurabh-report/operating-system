# Technical Journal Overview

This journal serves as the documentation for the **CMPN202 Operating Systems** coursework. It covers the following key areas:

1.  **System Planning**: Architecture design and OS selection.
2.  **Security**: Hardening, firewalls, and access control.
3.  **Performance**: Benchmarking and resource analysis.
4.  **Auditing**: Final system evaluation.

## Methodology

- **Server**: Ubuntu Server 24.04 LTS (Headless)
- **Host**: Windows 11 with PowerShell/SSH
- **Tools**: VirtualBox, OpenSSH, Git, Python, Stress-ng
